**PvRecNet** is a deep learning-based analytics platform designed for genomic classification of *Plasmodium vivax* recurrences. By using permutation-invariant set-processing architectures (Deep Sets), the system accurately distinguishes between the three biological pathways of recurrence: **Recrudescence**, **Relapse**, and **Reinfection**.

---

## 🧬 Core Methodology & Architecture

Differentiating true treatment failures from new mosquito inoculations or dormant liver stages is a major challenge in malaria surveillance. PvRecNet addresses this by training directly on the probabilistic outputs generated from the **[Pv3Rs](https://github.com/aimeertaylor/Pv3Rs)** framework. 

The online version of the model is hosted at **[https://huggingface.co/spaces/desinez/PvRecNet](https://huggingface.co/spaces/desinez/PvRecNet)**
